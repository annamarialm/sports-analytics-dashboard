# SportSee Frontend

Frontend application for the SportSee dashboard project.

## Stack

- React
- Vite
- React Router
- Axios
- Recharts
- Context API

## Launch

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

## Features

- JWT authentication
- Protected routes
- Dynamic dashboard
- Recharts visualizations
- Profile page
- Loading and error handling
- API integration with Axios
